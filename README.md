# The EasyShop E-Commerce API
This Capstone 3 Project is a simple e-commerce application.
## Features
### Phase 1: Categories Management
- View all categories
- Get Categories
- Create new categories (ADMIN only)
- Update existing categories (ADMIN only)
- Delete Categories (ADMIN only)

### Phase 2: Bug Fixes
- Product search filter bugs fixed
- Product update duplicating products instead of updating fixed

## Tech Used 
- Java 17
- MySQL Database
- Maven
- Spring Boot
- Spring Security with JWT
